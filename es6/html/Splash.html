FMS ACE

Improve Your Adminstration of FileMaker Server with FMS ACE

FMS ACE (FileMaker Server Admin Console Extension) is a JavaScript application (just like the FileMaker Server native Admin Console) that extends the native Admin Console provided by FileMaker, Inc.

The current version of ACE fetches all schedules from a FileMaker Server and displays them in a calendar view. It also allows you to create "verify database" and "send message" schedules.
